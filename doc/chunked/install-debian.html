Установка в Debian (Ubuntu)

Для начала нужно установить используемые пакеты сторонних производителей. Делается это в терминале командой:

sudo apt-get install -y python-yaml python-mutagen \
python-feedparser python-httplib2 python-webpy python-dnspython \
sox libsox-fmt-mp3 mp3gain vorbisgain icecast2 python-simplejson sqlite3 logrotate

После этого можно скачать и установить последнюю версию пакета ardj для Debian. Найти её можно на сайте ardj.googlecode.com:

http://code.google.com/p/ardj/downloads/list?q=label:OpSys-Debian

Если вы используете графическую оболочку — монжо открыть пакет файловым менеджером и следовать инструкциям. В терминале пакет устанавливается так:

sudo dpkg -i ardj_*.deb
