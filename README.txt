ARDJ is a software to operate an internet radio in automatic mode.  It
integrates with ices/ezstream, communicates with admins and listeners using
the XMPP protocol (Jabber), can send messages to twitter, download music from
Last.fm and Jamendo, and more.

Distributed under the GNU GPL.
