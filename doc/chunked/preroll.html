Подмотка

Подмотка (англ. preroll) — это композиция, автоматически вставляемая в плейлист перед композицией, которая была выбрана в соответствие с его настройками.  Подмотка используется для акцентирования внимания на конкретных композициях (например, если у исполнителя скоро концерт).

Есть два варианта использования подмотки.  Самый простой — подмотка для плейлиста: в свойстве preroll можно указать список меток, по которым будет выбираться подмотка для всех композиций плейлиста.

Второй вариант — подмотка на уровне меток, минуя плейлисты.  В этом режиме для каждой метки проигрываемой композиции разыскиываются композиции с меткой "метка-preroll".  На практике эти два метода всегда комбинируются.  Пример конфигурации плейлиста:

- name: rock
  preroll: [rock-jingle]

Если из этого плейлиста выбрана композиция с метками "rock", "music" и "male", для выбора подмотки будут использованы метки: "rock-jingle", "rock-preroll", "music-preroll", "male-preroll".  Случайная из найденных композиций будет проиграна перед запланированной.
